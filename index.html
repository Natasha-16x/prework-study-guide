HTML

  • HTML elements work together to build the structure of a webpage
  • HTML elements provide a framework that makes it easy for the end user to navigate the webpage
  • HTML elements are composed of an opening tag and a closing tag, represented with angle brackets
  • The head element contains information/metadata about the webpage.
  • The head element contains elements that are not visible when the user looks at the webpage
  • The body element represents the visble content shown to the user
  • The body element contains all elements we want the user to see and interact with

CSS

  • CSS is the language used to describe the presentation of a website that's created with HTML
  • CSS describes how each element on a page is rendered
  • CSS defines the style
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside and element

Git

  • A free, open source version control system
  • Allows web developers to collaborate on projects without overwriting each other's work
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • lightweight programming language that is used to provide logic and interactivity for websites
